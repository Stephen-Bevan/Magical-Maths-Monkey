The purpose of this website is to let the kids (6-8 year old) improve their multiplication skills (time tables). And help them to become more fluent and enjoy practicing maths. We are planning to extend the functionality of this game by using JavaScript in the next iterations - in order to make the game more dynamic and interactive for our learners.
